### For changes
1. Setup project
 * checkout the repository
 * run ```gradlew setupDevWorkspace```
 * (if using IntelliJ run ```gradlew genIntellijRuns```)

2. Do your changes ;)

3. Build
 * Use IDE or
 * ```gradlew build```

4. Use
 * Take the jar from ./build/libs/
